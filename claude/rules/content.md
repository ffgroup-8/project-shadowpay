# Content & SEO Rules

When writing content for PayNearMe pages:

SEO:
- Title format: Primary Keyword | PayNearMe (pipe separator, max 60 chars, never em dashes)
- Meta descriptions: max 155 chars, no em dashes, end with a soft CTA
- Schema types to consider: Organization, Service, FAQPage, WebPage, SoftwareApplication
- AEO targets: write as questions an AI assistant might ask (e.g. "How does PayNearMe work?")

Content boxes:
- seo-box = green box = metadata only (title, description, keywords, schema, AEO)
- content-created-box = blue box = actual copy modules only
- Purpose statement lives ABOVE both boxes as plain paragraph text
- Never put copy modules in the seo-box
- Never put SEO metadata in the content-created-box

Module copy must be real, finalized copy — not placeholders or layout notes.
